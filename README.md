---

Introduces two powerful and dangerous golden apple variants along with a utility command to reset player health. Designed for exploration rewards, server events, or admin tools, it provides a meaningful way to enhance your server's gameplay loop and challenge your players.

---

**Features**

*The Vital Apple*
* Permanent Health Boost - A rare and valuable item that permanently increases a player’s maximum health.
* Exploration Reward - Features a 10% chance to appear in naturally generated loot chests, granting a randomized boost of +1 to +3 hearts.
* Admin Command - Server operators can grant this item directly using `/giveauricapple <player> <hearts>`.

*The Cursed Death Apple*
* Permanent Health Reduction - A dangerous item that permanently reduces a player’s max health (down to a minimum of 1 heart).
* Punishing Effects - Inflicts instant damage, along with Blindness and Nausea for 3 seconds. It actively overrides typical golden apple effects like Regeneration and Absorption.
* OP Command Only - Perfect for hardcore challenges or risky PvP events. Obtainable only via the `/givecursedapple <player> <hearts>` command.

*Admin Tools*
* Health Reset - Safely restore a player’s max health back to the default 10 hearts (20 HP). Great for undoing changes or rebalancing gameplay after an event.
* Command - Use `/resethearts <player>` (OP only) to trigger the reset.
